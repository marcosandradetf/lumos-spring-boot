# 🚀 Primeiro Acesso ao Sistema

O acesso ao Lumos é realizado via navegador (Web) para gestores e via aplicativo (Android) para equipes de campo. Este guia foca no primeiro acesso ao **Painel Administrativo**.

---

## 🔐 Credenciais de Acesso

Ao ser cadastrado no sistema, você receberá um convite pelo Whatsapp ou por e-mail contendo:
1. **URL de acesso** (ex: `app.lumosip.com.br/primeiro-acesso`)
2. **Usuário** (CPF ou Primeiro Nome)
3. **Código temporário**

> ⚠️ **Segurança:** Neste primeiro acesso, o sistema exigirá obrigatoriamente o cadastro de uma senha pessoal forte.

---

## 🖥️ Requisitos do Sistema

Para uma experiência fluida, recomendamos:
- **Sistema Operacional** Windows 11 ou superior, macOS Sonoma ou superior, Android ou iOS.
- **Navegador:** Google Chrome ou Microsoft Edge (versões atualizadas).
- **Conexão:** Mínimo de 5MB para carregamento de dashboards e relatórios fotográficos.

---

## 🏁 Fluxo Inicial de Configuração

Para que o Lumos comece a gerar valor, recomendamos seguir esta ordem de configuração:

### 1. Perfil da Empresa
Acesse **Configurações > Empresa** para validar os dados da sua unidade, logotipo (que aparecerá nos relatórios) e fuso horário da operação.

### 2. Criação de Perfis de Acesso
Antes de convidar sua equipe, defina o que cada um pode fazer.
- *Exemplo:* O perfil "Eletricista" não deve visualizar valores de contratos.
- [Saiba mais sobre Perfis de Acesso](../02-access-management/01-profiles)

### 3. Cadastro do Primeiro Contrato
O Lumos orbita em torno do contrato. Sem ele, não há vínculo para as ordens de serviço.
- [Como cadastrar seu primeiro contrato](../04-contracts/01-setup)

---

## 📱 Acesso ao App de Campo (Android)

Para as equipes de campo:
1. Instale via Play Store 👉 [Guia de instalação](../03-operation/02-android-app-operational).
2. Utilize as mesmas credenciais criadas no painel web 👉 [Guia de primeiro acesso](../02-access-management/03-first-access-activation).
3. **Dica:** Realize o primeiro login em uma área com sinal de internet para sincronizar a base de dados inicial (Offline First).

---

## 🆘 Precisa de ajuda?

Se encontrar dificuldades no acesso:
- Verifique se o e-mail de convite não caiu na caixa de **Spam**.
- Entre em contato com o administrador do sistema na sua empresa.
- Ou envie um e-mail para: **suporte@lumosip.com.br**
