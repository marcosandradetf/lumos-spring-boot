# ➕ Convidando Usuários

No Lumos, o acesso dos colaboradores começa com um **convite seguro**, sem envio de senhas.

O administrador cria o usuário e gera um **código de ativação temporário**, que será utilizado no primeiro acesso.

---

## 👤 Cadastrando um Novo Usuário

1. No Painel Web, acesse **Configurações > Usuários**.
2. Clique em **Novo Usuário**.
3. Preencha os dados:
   - **Nome Completo**
   - **CPF**
   - **Perfil de Acesso**
4. Salve o usuário.

Após o cadastro:
- O usuário será criado com status **“Pendente de Ativação”**
- Ele ainda **não poderá acessar o sistema**

---

## 🔑 Gerando o Código de Ativação

1. Localize o usuário na lista
2. Clique em **Gerar Código de Ativação**

O sistema irá gerar um código temporário, que deve ser enviado ao colaborador.

---

## 📤 Compartilhando o Código

Você pode compartilhar o código de duas formas:

- 📲 **Enviar via WhatsApp** (recomendado para equipes de campo)
- 📋 **Copiar e enviar manualmente**

A mensagem enviada já contém instruções de acesso para facilitar o processo.

> ⚠️ **Importante:** O código possui validade limitada e expira automaticamente por segurança.

---

## 🔄 Regerar Código de Ativação

Se o código expirar ou o usuário não tiver recebido:

1. Vá em **Configurações > Usuários**
2. Selecione o usuário
3. Clique em **Regerar Código**

> 🔁 O código anterior será invalidado automaticamente

---

## 🧠 Dica de Gestor

Sempre gere o código próximo ao momento em que o colaborador for acessar o sistema.

Isso evita:
- expiração do código
- retrabalho
- dificuldades no onboarding
