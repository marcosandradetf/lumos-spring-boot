# 🔐 Primeiro Acesso e Ativação

Após receber o código de ativação, o colaborador deve realizar o **primeiro acesso** para liberar o uso do sistema.

Esse processo garante segurança e evita o compartilhamento de senhas.

---

## 📱 Ativação no Aplicativo (Campo)

1. Abra o aplicativo Lumos OP
2. Toque em **“Primeiro Acesso”**
3. Informe:
   - **CPF**
   - **Código de Ativação**
   - **Nova Senha**
   - **Confirmação da Senha**
4. Toque em **Ativar Conta**

Após isso:
- A conta será ativada
- O acesso estará liberado

---

## 💻 Ativação na versão Web

1. Acesse por esse link 👉 https://app.lumosip.com.br/primeiro-acesso
2. Ou acesse a tela de login e clique em **“Primeiro Acesso”**
3. Preencha:
   - CPF
   - Código de ativação
   - Nova senha
4. Confirme

---

## 🚫 Acesso Antes da Ativação

Se o usuário tentar entrar no sistema sem ativar:

- O acesso será bloqueado automaticamente
- Ele será direcionado para concluir o primeiro acesso

---

## 🔒 Regras de Segurança

- Senhas não são compartilhadas nem visíveis
- O código de ativação:
  - ⏳ Expira automaticamente
  - 🔢 Possui limite de tentativas
- O acesso só é liberado após ativação

---

## ⚠️ Problemas Comuns

**Código expirado?**  
→ Solicite um novo código ao administrador

**Código inválido?**  
→ Verifique se foi digitado corretamente

**Não recebeu o código?**  
→ Entre em contato com o gestor responsável

---

## 🧠 Dica para o Usuário

Realize o primeiro acesso assim que receber o código.

Isso garante:
- ativação rápida
- menos risco de expiração
- acesso imediato ao sistema
